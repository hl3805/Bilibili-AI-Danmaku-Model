# 子任务 G：标题范式提取与流量转化效能检验

## 1. 使用数据
* `cleaned-data/bilibili_video_cleaned.csv`

## 2. 数据处理指引 (Agent 自主选择路径)
* **模板特征提取 (NLP)**：
  * 简单路径：使用正则表达式捕捉特征符号（如 `【.*?】`、`！`、`？`）。
  * 深度路径：使用依存句法分析（Dependency Parsing）提取句型骨架（如 "动词+名词+从入门到放弃"），或对标题文本向量进行 K-Means 聚类，然后人工归纳出 K 个簇代表的模板（如：教程型、盘点型、情绪型）。
* **消除基数偏差**：依然使用相对流量（单视频播放量 / 粉丝数）或进行倾向得分匹配（PSM）后对比。

## 3. 分析与验证分支逻辑
* **检验假设**：存在某些特定的标题模板（如“情绪渲染型”），其相对转化效能显著优于其他模板。使用单因素方差分析（ANOVA）加 Tukey 事后检验。
* **分支 A（模板间存在显著鸿沟）**：特定范式的流量效率是其他范式的数倍。
* **分支 B（模板间无显著差异，方差全在 UP 主体量内）**：无论用什么模板，起决定性作用的只剩下 UP 主本身的 IP 效应。

## 4. 业务洞察推演
* **基于分支 A**：整理《AI 领域高转化标题范式指南》，作为给中腰部 UP 主直接可用的实操物料。提示哪类旧模版（如古典的“震惊体”）已经因用户免疫而数据崩盘。
* **基于分支 B**：破除“标题迷信”。向创作者传达：在硬核知识区，不要试图通过标题的文字游戏逆天改命，建立粉丝对 IP 的专业度信任才是唯一解。

## 5. 局限性与未来深化方向
* **视觉信息的剥离**：B 站的点击决策是“封面 + 标题”共同决定的。由于我们缺乏封面图（Cover Image）数据，标题的作用可能被高估或低估。未来方向必须引入计算机视觉（CV）模型分析封面色彩、大字报与标题的耦合效应。
