# B站AI主题视频数据采集报告

## 一、采集规则

### 1.1 目标标签

| 主标签 | 匹配方式 | 说明 |
|--------|----------|------|
| AI焦虑 | 模糊匹配 | 含"焦虑"/"失业"/"取代工作"等关键词 |
| 科技/人工智能 | 精确+模糊 | 科技大类下AI相关内容 |
| 人工智能 | 精确+模糊 | AI相关内容 |
| DeepSeek | 模糊匹配 | 含"DeepSeek"/"DeepSeek教程"/"AI大模型"等 |

### 1.2 时间范围

- **2024年-2025年8个季度** 
- 每季度采集10页（科技tag5页）
- 按发布时间分页

### 1.3 过滤条件

```
播放量 ≥ 50,000
弹幕数 ≥ 50
```

### 1.4 采集策略

- 并发数：3
- 基础延迟：1-2秒（自适应）
- 错误延迟：2-4秒
- IP限制冷却：60秒

## 二、视频数据字段

| 字段 | 说明 |
|------|------|
| bvid | 视频唯一ID |
| aid |  avid编号 |
| cid | 弹幕cid |
| title | 视频标题 |
| pubdate | 发布时间 |
| tname | 分区名称 |
| owner_mid | UP主ID |
| owner_fans | UP主粉丝数 |
| duration | 视频时长(秒) |
| desc | 视频描述 |
| stat_view | 播放量 |
| stat_danmaku | 弹幕数 |
| stat_reply | 评论数 |
| stat_like | 点赞数 |
| stat_coin | 投币数 |
| stat_favorite | 收藏数 |
| stat_share | 分享数 |
| tags | 视频标签(竖线分隔) |
| search_tag | 采集所用搜索标签 |
