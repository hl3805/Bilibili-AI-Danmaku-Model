
# 第0步：文献回顾与相似项目调研 (Phase 0)

## 一、 任务概述
在正式开展数据采集和分析之前，Agent需要通过网络搜索、GitHub仓库检索以及学术文献查找，完成关于“B站视频数据分析”、“视频标签(Tag)对流量的影响”、“弹幕文本NLP分析（受众泛化）”以及“百度指数与社交媒体的时序相关性”的相似项目和前沿文献调研。

## 二、 调研执行要求
你需要查找并总结至少 **4-5 个** 相关度高的项目或学术论文。这些资料应涵盖以下至少两个维度：
1. **平台与指标**：基于B站或抖音等视频平台的推荐算法与流量归因分析。
2. **内容特征**：研究视频标签（Tag）、标题（Title）如何影响算法推荐池或用户点击。
3. **时序与外部热度**：搜索引擎指数（如Baidu/Google Trends）与社交平台内容发布量的超前/滞后关系。
4. **互动与NLP**：利用LDA主题模型或情感分析研究视频生命周期内的弹幕/评论演变趋势。

## 三、 输出格式标准
在完成调研后，请在 `results/` 目录下生成一个名为 `related-work.md` 的文档。对于你找到的每一个项目/论文，**必须严格按照以下结构进行总结**：

### 示例模板（对于每个调研项目）：

#### [项目/论文标题名称]
* **1. 问题/假设**
  * 该研究试图解决什么核心业务问题或学术问题？（例如：长尾标签是否比热门标签带来更高的转化率？）
  * 提出了哪些关键假设？
* **2. 解决方案/方法**
